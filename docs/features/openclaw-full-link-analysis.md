## 模块概要介绍

&#x20;

OpenClaw 调用链分析模块提供面向运维与开发的分布式系统全链路追踪能力，完整采集、存储与分析 OpenClaw 平台微服务架构下的所有调用数据。整合了请求执行路径、性能指标、异常信息、Token 消耗与业务属性数据，提供从全局集群概览到单实例监控、再到单 Trace 全链路追溯的多层级分析能力，支持多维度筛选检索、性能瓶颈定位、异常根因诊断与成本精细化管控，帮助用户快速解决分布式系统中的问题，提升平台稳定性与资源使用效率。

## 调用链概览（全局驾驶舱）

&#x20;

提供平台整体调用链运行状态的全局可视化监控视图，通过多维度指标聚合与趋势分析，帮助用户快速掌握系统整体健康度与运行趋势。

- 支持按时间维度与粒度筛选，所有图表与指标实时联动更新，帮助用户识别系统流量波动、性能异常与故障扩散趋势，为系统性问题排查提供数据支撑。
- 时间范围筛选：支持「最近 1 小时 / 6 小时 / 12 小时 / 24 小时 / 3 天 / 7 天」快速切换，也可自定义精确时间范围，支持 1 分钟粒度数据聚合，适配不同周期的监控分析需求。
- 核心指标聚合：直观展示总请求次数、成功次数、失败次数、整体成功率、平均耗时与最大耗时，同时展示各指标的环比变化率与关键异常统计（失败占比、成功率 <95% 实例数、耗时> 10s 实例数），快速感知系统运行状态变化。
- 实例趋势分析：按时间展示调用量趋势、成功率趋势、平均 / 最大耗时趋势与异常次数趋势四个独立图表，多维度呈现系统运行波动。
- 实例分析：通过两个表格分别展示各实例的调用次数与失败占比、最大耗时与成功率，快速识别异常实例。
- TOP 排行分析：按最大耗时、最低成功率、失败次数三个维度展示接口排行，支持 Top5/Top10/Top20 切换，聚焦系统性能最差与最不稳定的接口。
- 业务与性能分析：展示不同消息渠道的调用次数与成功率、各模型 Token 消耗排行与请求耗时分布，关联业务属性与技术指标。

## 调用链实例列表

提供所有调用链实例的集中式列表监控能力，支持多维度筛选、检索与状态监控，实现对所有集群实例的高效管控。

- 提供实例状态聚合与多条件组合筛选功能，帮助用户快速过滤异常实例，适配大规模集群的运维管理需求。
- 核心指标聚合：展示实例总数、运行中 / 异常 / 离线实例数、集群总调用次数（含环比变化率）、集群平均成功率（含成功率 <95% 实例数）与集群平均响应时（含耗时> 10s 实例数），同时标注异常实例数量，快速感知集群整体状态。
- 多维度筛选检索：支持关键词模糊搜索（实例 ID / 主机名 / IP 地址），以及实例运行状态、部署环境组合筛选，快速定位目标实例。
- 实例列表详情：集中展示实例基础信息（实例 ID、主机名称、主机 IP 地址、运行状态、部署环境）、执行统计（总调用次数、失败次数、成功率、平均耗时、最大耗时、总 Token 消耗），异常数据以红色高亮显示。
- 状态标识：运行正常（绿色）、异常告警（黄色）、严重异常（红色）、离线（灰色），直观区分实例运行状态。
- 操作与控制：支持列设置自定义显示字段，点击 "查看详情" 跳转至单实例深度分析页面，支持分页加载与每页显示条数调整，底部展示数据范围与总条数，适配大规模实例场景。

## 实例详情（单实例深度分析）

&#x20;

提供单个调用链实例全方位深度分析能力，整合概览监控、多维度查询与专业分析工具，以时间维度聚合调用数据、性能指标与异常信息，支持多时间范围筛选，适配不同排查与分析需求，数据随时间联动更新。

- 页面头部与基本信息：提供实例基础信息与核心指标聚合视图，支持返回列表及「近 10 分钟 / 30 分钟 / 1 小时 / 6 小时 / 24 小时 / 7 天」时间筛选；展示实例基础信息、核心运行指标，双 Y 轴调用趋势图及多条件高级筛选栏，精准定位目标数据。
- &#x20;

  核心指标与分析图表：展示实例关键性能指标，通过 Span 类型分布、耗时堆叠图、渠道调用堆叠趋势等图表，助力评估性能、识别异常与了解业务流量分布。
- Span 列表：集中查询实例下所有 Span，展示 TraceID、接口名称等详细信息，支持搜索、排序、分页，可跳转至 Trace 详情页。
- Trace 列表：以 Trace 为维度展示请求整体情况，包含 TraceID、入口接口等信息，支持搜索、排序、分页，可跳转至 Trace 详情页。
- 散点图分析：以散点图展示请求耗时分布，蓝点代表正常请求、红点代表异常请求，提供 5s/10s 参考线，支持异常筛选及 TOP20 异常 Trace 跳转。
- Apdex 分析：支持自定义 Apdex 阈值，展示实例整体 Apdex 指数、请求占比、变化趋势及接口排行，量化用户体验。
- 全链路聚合：支持按多维度聚合调用数据，可选择多种核心指标展示，搭配聚合趋势图，帮助宏观了解系统运行情况。
- 全链路拓扑：以拓扑图展示服务间调用关系与流量分布，节点大小、连线粗细对应调用量与流量，异常节点及连线高亮，支持缩放、平移与详情查看。

## Span 列表

&#x20;

提供实例维度下全量调用 Span 的集中式查询与管理能力，完整记录每次调用的基础信息与执行状态，支持多维度筛选检索，帮助用户快速定位异常调用节点。

支持列表排序、分页与详情跳转，实现从 Span 到完整 Trace 链路的一键追溯，提升问题排查效率。

- 全局筛选联动：继承实例页时间范围与高级筛选条件，支持状态（正常 / 错误）、耗时范围、接口名称、消息渠道、模型名称与会话 Key 的组合筛选，精准过滤目标数据。
- 搜索与排序：支持按 TraceID、接口名称、会话 Key 全局模糊搜索，所有列表列支持升序 / 降序排序，快速定位异常调用。
- 列表字段展示：包含 TraceID、接口名称、应用名称、调用类型、父 SpanID、会话 ID、耗时、状态、开始时间、主机地址，错误状态与异常耗时以红色高亮显示。
- 分页与操作：支持分页加载与每页显示条数调整，每条记录提供 "查看详情" 按钮，点击直接跳转至对应 Trace 详情页并自动定位到该 Span。

# Span 详情

提供单个调用 Span 的全维度详情展示能力，整合基础信息、性能指标、业务属性与异常上下文，完整还原单次调用的执行过程与问题细节。

与火焰图联动展示，支持节点快速定位与上下文查看，帮助用户精准定位调用异常的根本原因。

- 基础信息展示：呈现应用名、接口名称、主机 IP 地址、开始时间、SpanID、父 SpanID、调用类型与运行状态，明确调用的基本上下文。
- 性能上下文分析：展示该 Span 的实际耗时、所属 Trace 的平均耗时、耗时分位值与耗时分布区间，量化调用性能表现。
- OpenClaw 业务属性：关联展示会话 ID、运行 ID、轮次 ID、版本号、调用渠道等业务标识，打通技术指标与业务场景。
- 异常信息详情：异常 Span 以红色高亮展示，包含异常类型、错误代码与完整错误堆栈信息，直接定位问题根源。
- 全量属性展示：提供 Attributes 自定义属性面板，展示该 Span 的所有扩展字段与标签信息，提供完整的排查上下文。

# Trace 列表

提供实例维度下全量请求 Trace 的集中式查询与管理能力，以请求为维度聚合展示整体执行情况，帮助用户从宏观视角识别异常请求。

支持多条件筛选与排序，实现从请求概览到完整链路的一键跳转，提升大规模请求下的问题排查效率。

- 全局筛选联动：继承实例页时间范围与高级筛选条件，支持按运行状态、耗时范围、入口接口等进一步过滤，精准定位目标请求。
- 搜索与排序：支持按 TraceID、入口接口名称全局模糊搜索，支持按总耗时、开始时间、Span 数量等多维度排序。
- 列表字段展示：包含 TraceID、入口接口、Span 数量、总耗时、运行状态、开始时间，异常 Trace 与高耗时 Trace 以红色高亮显示。
- 分页与操作：支持分页加载与每页显示条数调整，每条记录提供 "查看详情" 按钮，点击直接跳转至对应 Trace 的完整详情页面。

# Trace 详情

提供单个请求的完整全链路追溯能力，通过火焰图可视化与多维度详情面板，精准还原请求从入口到结束的完整执行路径、耗时分布与异常节点。

整合可控成本分析能力，实现技术指标与资源成本的联动追溯，帮助用户同时解决性能问题与成本优化问题。

- 页面头部与操作栏：展示核心链路指标（应用数、接口数、检测到的异常数、Spans 总数、总耗时、错误数），提供全局搜索、展开全部、折叠全部、复制 TraceID 等快捷操作。
- 火焰图可视化：以层级结构展示完整调用链路，矩形长度对应 Span 耗时，正常 Span 为蓝色、异常 Span 为红色高亮；支持缩放、平移、节点展开折叠，点击节点自动联动右侧详情面板。
- 多维度详情面板：分模块展示选中 Span 的基础信息、性能上下文、OpenClaw 业务属性、异常信息与全量自定义属性。
- 可控成本分析：提供下拉菜单切换查看成本概览、会话成本明细、Agent 成本明细与模型成本明细，在火焰图中同步展示各节点的成本占比，实现全链路成本追溯。

## 散点图分析

&#x20;

提供实例下全量请求的耗时分布可视化视图，通过散点直观呈现请求耗时与异常分布，帮助用户快速定位慢请求、异常请求与性能瓶颈。

支持按 Y 轴指标切换、异常筛选，图表与下方异常列表联动，精准锁定问题请求。

时间范围筛选：继承实例页时间筛选条件，适配不同周期的耗时分析需求。

散点分布视图：X 轴为请求时间、Y 轴为总耗时，蓝色点代表正常请求、红色点代表异常请求，配置 5s、10s 两条耗时参考线，直观识别超限请求。

筛选控制：支持 Y 轴指标切换（总耗时 / 平均耗时）、仅展示异常请求筛选，精准过滤目标数据。

TOP 异常 Trace 列表：展示耗时最长的 TOP20 异常 Trace，包含 TraceID、入口接口、总耗时等信息，点击可跳转至 Trace 详情页定位根因。

## Apdex 分析

通过 Apdex 应用性能指数量化实例的用户体验与服务质量，从满意度维度评估系统性能表现，助力性能优化与稳定性提升。

支持自定义性能阈值，多维度呈现 Apdex 指数、请求占比、趋势与接口排行，精准定位性能短板。

阈值配置：支持自定义 Apdex 响应时间阈值 T，自动计算满意、可容忍、不满意请求区间。

核心指标聚合：直观展示实例整体 Apdex 指数、满意请求占比、可容忍请求占比、不满意请求占比，快速评估服务体验等级。

Apdex 趋势图：按时间展示 Apdex 指数变化趋势，识别性能波动与周期性问题。

接口 Apdex 排行：按接口维度展示 Apdex 指数、请求占比、总调用次数、平均耗时，精准定位性能较差的接口。

## 全链路聚合

提供实例调用数据的多维度聚合分析能力，从宏观维度汇总调用指标，实现批量接口的性能、成功率、资源消耗对比分析。

支持自定义聚合维度与指标，搭配趋势图直观展示指标变化，为批量优化提供数据支撑。

聚合维度选择：支持按接口名称、应用名称、主机地址等维度聚合调用数据。

指标筛选：支持勾选调用次数、成功率、平均耗时、P99 耗时、失败次数、总 Token 消耗等多类指标。

聚合结果表格：展示各维度下聚合后的指标数值，横向对比不同对象的运行表现。

聚合趋势图：按时间展示多接口核心指标的变化趋势，直观识别接口性能波动与流量变化。

## 全链路拓扑

以可视化拓扑图展示实例内全量接口的调用依赖关系、流量分布与异常链路，清晰呈现系统架构与问题传播路径。

支持聚合参数配置、缩放交互，异常节点与链路高亮，快速识别故障链路与性能瓶颈。

聚合参数配置：支持按接口维度聚合，自定义聚合 Trace 数量，保证链路聚合完整性。

拓扑节点展示：节点代表接口 / 服务，展示调用次数、错误数、请求数、响应时间，节点大小对应调用量。

拓扑连线展示：连线代表调用关系，标注调用占比，异常链路以红色高亮，直观识别故障传播路径。

交互功能：支持缩放、平移、节点详情查看，快速定位异常节点与上下游依赖。
